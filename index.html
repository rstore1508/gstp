Login Não Ativado

O Firebase bloqueou a conexão porque o Login Anônimo não está ativado.

  1. Vá ao Firebase Console > Authentication.
  2. Aba Sign-in method.
  3. Ative o provedor Anonymous.

Permissão Negada

O banco de dados está bloqueado. Você precisa alterar as Regras de Segurança para permitir leitura e escrita.

Como resolver:

  1. Vá ao Firebase Console.
  2. Clique em Firestore Database > Aba Rules.
  3. Substitua allow read, write: if false; por:
  4. allow read, write: if request.auth != null;
  5. Clique em Publish.
